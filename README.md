# Fundamental Booster Python Program

This program is a simple Python script that demonstrates basic input/output operations, type conversion, and variable usage.

## Features
- Prompts the user to enter their name, age, and height.
- Calculates the user's birth year based on the current running year (set to 2026).
- Displays the user's name, age, and height.
- Demonstrates type conversion between `int` and `float`.
- Shows how to use the `type()` and `id()` functions in Python.

## How to Run
1. Make sure you have Python installed on your system.
2. Open a terminal or command prompt.
3. Navigate to the directory containing the script.
4. Run the script using:
	```
	python "fundamental booster.py"
	```
5. Follow the prompts to enter your name, age, and height.
